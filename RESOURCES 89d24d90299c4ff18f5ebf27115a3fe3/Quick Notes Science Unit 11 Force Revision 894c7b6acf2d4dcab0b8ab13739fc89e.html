Basic ideas of force

  • Force is a push or a pull of object
  • We represent force using arrow
  • The arrow is drawn from where the force is applied and it points in the direction of force
  • The longer the arrow, The larger the force.

Effect of force

It change the motion of object

  • It change the speed of an object
    • It change moving object to move faster or slower
    • It make the object at rest (stationary object) to start moving
    • It make the moving object to rest
  • It change the direction of moving object

Motion

  • Speed
    • The speed of an object is the distance of a object moved in a period of time.
    • It unit can be ms−1m s^{-1}ms−1﻿ and kmkm km﻿h−1h^{-1}h−1﻿.
    • The faster the object, the higher its speed
    • Average speed = distance / time
    • We need to write the formula of calculate the average speed of an object
  • Distance-time graph
    • We use the distance time graph to show the motion of an object.
    • We use a distance-time graph to show the distance that an object has travelled at different times
    • When the object is at rest, the distance-time graph show a horizontal line
    • When the object is moving at a steady speed, the distance-time graph is a straight line and with a slope. The faster the speed of the object, the steeper the slope.
    • When the object is moving in a non-uniform motion, the distance-time graph show a curve line
    • From time = ___ to ___, the object travelled in a uniform motion / non-uniform motion.
      • The object is move in a steady speed / coming to rest / at rest. It move ____ (m).
      • The object move in a slower / faster speed.
  • Uniform motion / non-uniform motion
    • Uniform motion is the object move in a constant speed and in a fixed direction.
    • Non-uniform motion is the object move in a changing speed or direction.

Measuring force

  • Force can be measured by spring balance. It can only measure the pulling force
  • The force can also measured by force sensor connected to data logger. It can measure both pushing and pulling force.

Contact forces and non-contact forces

  • Contact Forces act on an objects only when the objects are in contact
    • E.g. Supporting Force
    • E.g. Friction and and air resistance

Friction

  • When an object slides or tends to slide over a surface, friction acts on the object and opposes the object’s sliding motion.
  • Friction opposes the direction of sliding motion of an object over a surface it is in contact with
  • Static friction
    • When apply a force on an object and the object not move, friction still exists called static friction.

Balanced forces and unbalanced forces

When the forces act on the object can be perfectly balanced by other forces with same magnitude and opposite direction called balanced force. balanced force = zero net force

When the balanced force act on a moving object, the object will move in a steady speed.

When the balanced force act on a object in rest, the object will still in rest.

When the unbalanced force act on a object, it will change the motion of a object.

Action and Reaction

Forces occur in pairs and these pairs of forces are called action-and-reaction pairs.

The action and reaction forces are in same magnitude and opposite direction.
