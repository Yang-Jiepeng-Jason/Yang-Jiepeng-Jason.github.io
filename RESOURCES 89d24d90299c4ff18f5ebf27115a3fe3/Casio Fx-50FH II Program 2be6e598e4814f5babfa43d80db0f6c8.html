First Version v1.0

This program is to automatically convert decimal numbers to different bases.

This Program solve many shortcomings of the built-in BASE MODE of the calculator itself.

This program contain 93 bit and run under COMP MODE.

Shortcomings of the built-in BASE MODE

  • It cannot input some formulas with powers and functions. Such as: √x,ab,10x,x!√x, a^b, 10^x, x!√x,ab,10x,x!﻿
  • For decimal to binary numbers, due to the limitation of displaying digits, they can only convert up to 51210512_{10}51210​﻿.

Shortcomings of the Program

  • Slow computation speed
  • Can only convert the decimal numbers to other bases

Advantages of the Program

  • It CAN input the formulas with powers and functions. Such as: √x,ab,10x,x!√x, a^b, 10^x, x!√x,ab,10x,x!﻿
  • The maximum input limit is up to 523287.
